What

What is Signal Gap Mesh?

Signal Gap Mesh is a community-based, open-source, off-grid, decentralized, LoRa mesh network for text messaging.

The platform enables individuals to transmit unencrypted group text messages to all network participants, and users can also send encrypted messages to any specific member.

All of this occurs without any external infrastructure: no cell phone towers, no internet, and, in some cases, no external power source. Just radio waves!

What is Mesh network?

Mesh network is a type of wireless communication network where individual devices can act both as endpoints and as relays to extend the network's range and coverage.

In a traditional point-to-point communication, devices directly communicate with a central gateway. In a mesh network, however, devices can communicate with each other and relay data to reach distant devices. This enables the network to cover larger areas and overcome obstacles that could otherwise block direct communication.
